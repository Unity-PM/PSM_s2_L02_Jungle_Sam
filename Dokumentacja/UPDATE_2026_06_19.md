## 4. MutantStalker AI / Animator Notes

`MutantStalkerAI`:
- uses `NavMeshAgent`,
- finds player by `Player` tag,
- implements `IDamageable`,
- supports chase, attack, hit reaction, rage and death,
- uses action locks to stop the agent during attack/gethit/rage.

Action lock values:

```text
Attack Animation Lock: 1.1
Hit Reaction Lock: 0.28
Hit Reaction Cooldown: 0.35
Rage Animation Lock: 1.2
```

`MutantStalkerAnimator`:
- uses `Animator.StringToHash`,
- keeps 1-based indices,
- blocks locomotion crossfade during action states,
- ignores new animation calls after death.

## 5. Damage System

Added:

```text
Assets/JungleSam/Scripts/Combat/IDamageable.cs
```

`MutantStalkerAI` implements `IDamageable`.

`WeaponBase` now checks:

```csharp
IDamageable damageable = hit.collider.GetComponentInParent<IDamageable>();
```

If no `IDamageable` is found, it falls back to old `EnemyAI` damage logic.
